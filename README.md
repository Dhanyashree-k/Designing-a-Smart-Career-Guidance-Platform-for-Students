# Designing-a-Smart-Career-Guidance-Platform-for-Students
This repository contains the complete UX/UI design process and high-fidelity prototypes for a **Career Guidance Mobile App** created in Figma.
The project helps students, job seekers, and career changers discover suitable careers through a structured quiz, personalized recommendations, dashboards, and AI mentorship.
